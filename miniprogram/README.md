# 阳明心学交易系统小程序 MVP

这个目录是微信小程序第一阶段 MVP，定位为“每日修行器”：

- 首页：今日心证、今日一省、快捷入口
- 照见：九型人格照见
- 心证：主/次人格、触发点、行为偏差、事上练路径
- 事上练：每日修心事上练卡、三步打卡、省察一句
- 省察：AI省察、AI心证、知行合一分数、明日克治动作
- 六大关卡：立志、照心、事上练、破心中贼、知行合一、致良知
- 知行成长记录：近7天、近30天知行指数与修行等级
- 成长树：连续修行、关卡进度、道场任务反哺
- 道场：阳明先生作为道统导师，真人带练称为修行营教练或助教，包含共修任务、AI观心助手、交易人格心证入口、本地修行榜
- 我的：积分、连续打卡、资料领取、助理承接、合规边界

## 本地预览

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本目录：`miniprogram`。
4. AppID 可先使用游客模式或微信开发者工具体验号。
5. 编译后从 `pages/home/index` 进入。

## 字体系统

小程序已内置裁剪后的字体子集：

- 心证、戒律、六大关卡、九型人格名称：`ZX-LXGW`。
- 正文说明、AI省察、数据面板：`ZX-Harmony` Regular / Medium。
- 知行指数数字、成长数值、连续修行天数：`ZX-Harmony` Bold。

原始字体文件较大，不直接放入小程序包；后续新增大量文案时需要重新生成字体子集。

## 后端同步

小程序采用“本地优先 + 后端同步”：

- 默认后端地址：`http://127.0.0.1:8787`
- 开发者工具模拟器可直接使用默认地址。
- 手机真机预览时，进入“我的”页，把后端地址改成电脑局域网 IP，例如：`http://192.168.1.8:8787`。
- 后端需先在 `server` 目录启动：`npm run dev`。

当前同步内容：

- 账号：通过后端 `POST /api/v1/auth/demo-login` 生成小程序体验账号。
- 心证：本地照见结果与答案同步到 `miniprogram-state`。
- 事上练：每日事上练卡进度、省察一句同步到后端。
- 省察：收盘省察字段、AI省察、AI心证、知行分数、明日克治动作同步到后端。
- 修行快照：`mind_profile` 保存当前人格、关卡、心贼、戒律、事上练动作。
- 知行成长：`zhixing_score` 保存每日知行指数历史、最新分数和成长阶段。
- 道场：道场码、修行营教练/助教关系、共修任务、观心记录同步到后端。
- 打卡：完成每日事上练时同步调用 `POST /api/v1/users/:user_id/check-in`。

同步接口：

- `GET /api/v1/users/:user_id/miniprogram-state`
- `POST /api/v1/users/:user_id/miniprogram-state`

## 合规边界

所有页面保留统一声明：

本系统仅用于交易认知、事上练与风险教育；不推荐股票、不提供操作指令、不承诺结果。

第一阶段不接实时行情，不做交易功能，不做复杂后台。后续可接当前工作区内的后端 API，用于账号、心证、事上练记录、省察记录同步。
