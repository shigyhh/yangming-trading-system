# 阳明心学交易修行系统：世界观与系统规则

## 一句话

我们不是帮助用户预测市场，而是帮助交易者在市场中照见自己。

本系统不是行情软件、选股工具、投顾服务、社区论坛，也不是收益展示平台。它是交易者每日进入的修行空间，训练对象不是市场判断，而是知行合一。

## 核心闭环

所有功能必须服务于这条闭环：

人格识别 -> 主修关卡 -> 每日训练 -> 知行指数 -> 持续成长

如果一个功能不能推动用户进入下一步修行，就应当合并、弱化或删除。

## V1 固化范围

V1 阶段暂停新增页面，只固化六个产品内核：

1. 六大关卡体系。
2. 九型人格与关卡映射。
3. 知行指数算法。
4. 365天心证库数据结构。
5. 每日训练卡数据结构。
6. AI观心教练接口预留。

任何改动都必须服务于：

人格识别 -> 主修关卡 -> 每日训练 -> 知行指数。

V1 的技术实现必须有一个统一成长引擎，页面只读取成长结果，不各自决定主修关卡、今日训练和知行指数。

## 禁止边界

禁止在产品前端、直播话术、AI回应、报告和运营后台中出现以下导向：

- 推荐具体买卖
- 展示具体股票代码和结果截图
- 承诺收益或暗示确定性结果
- 以收益、交易战绩、账户金额作为排行榜依据
- 让训练营教练或助教查看用户具体交易明细

禁用词：

- 暴富
- 翻倍
- 妖股
- 龙头
- 必赢
- 稳赚
- 财富自由

## 九型人格规则

九型人格不是风险偏好，也不是投资能力评价。它只用于识别交易者最容易偏离计划的心性惯性，并驱动每日训练内容。

| 人格 | 核心惯性 | 心贼 | 戒律 | 每日训练 |
| --- | --- | --- | --- | --- |
| 冲动型 | 见动即动，行动先于计划 | 怕错过 | 无计划，不开仓 | 行动前写下理由、边界、离场条件 |
| 扛单型 | 明知边界已到，仍想再等 | 不甘认错 | 触发边界，不移动 | 触发边界后只执行预案 |
| 完美型 | 把复盘变成自责 | 怕犯错 | 一错一改，不责其心 | 每次复盘只写一个可修正动作 |
| 赌徒型 | 越乱越想用更大的动作修复情绪 | 不甘和侥幸 | 赌念一起，先降一档 | 想搏一下时立刻记录，不追加动作 |
| 从众型 | 外部声音越大，自己的计划越小 | 怕独自判断 | 众声入耳，先问本心 | 盘中只看自己的计划卡 |
| 偏执型 | 只看支持自己判断的证据 | 证明自己 | 先格物，再立言 | 行动前写一条反向证据 |
| 拖延型 | 知道要复盘，但总想明天再做 | 逃避真实记录 | 今日事，今日省 | 三分钟写下触发、反应、明日修正 |
| 焦虑型 | 用高频关注换安全感 | 惧念 | 心乱时，动作要更少 | 固定观察窗口，窗口外只记录心境 |
| 平衡型 | 稳定时容易省略基本功 | 轻忽 | 稳定时，更要守一 | 保留每日一省 |

## 六大关卡规则

六大关卡是成长树的唯一主轴。任何训练、任务、复盘和道场行为，都应该反哺其中一关。

详细定义以 `worldview/six-stages.md` 为源文件。小程序、官网、直播内容、训练卡和后续AI提示词都必须服从这个定义。

| 关卡 | 目标 | 完成标准 | 系统输出 |
| --- | --- | --- | --- |
| 立志 | 建立交易戒律 | 有交易计划、有仓位规则、有止损规则 | 今日戒律、行动边界、立志记录 |
| 照心 | 识别情绪 | 能识别贪、能识别惧、能识别急 | 今日心境、今日禁忌、今日照心提醒 |
| 事上磨 | 执行纪律 | 止损执行、不追涨、不扛单 | 每日训练卡、完成记录、主修兑现分 |
| 破心贼 | 克服人格惯性 | 识别主人格、修正主人格 | 当前人格、当前心贼、主修关卡、今日戒律、今日训练 |
| 知行合一 | 计划与执行一致 | 连续执行、不临盘改计划 | 知行指数、纪律执行分、系统一致性分、临盘克制分 |
| 致良知 | 形成成熟交易人格 | 自主复盘、自主觉察、长期稳定 | 成长阶段、知行成长曲线、长期修行路径 |

关卡判定原则：

- 人格决定主修关卡。
- 主修关卡决定今日训练。
- 今日训练决定知行指数的主修兑现。
- 知行指数反过来更新关卡进度。

## 知行指数规则

知行指数不是收益分，不是交易能力分，也不是排名炫耀分。它衡量用户今天是否把说过的计划变成行动。

总分由五个维度构成：

| 维度 | 衡量对象 | 主要输入 |
| --- | --- | --- |
| 纪律执行 | 是否按预设边界执行 | 是否系统内、是否执行边界、是否临盘改计划、今日修行完成度 |
| 情绪稳定 | 是否被贪、惧、急、不甘牵动 | 今日照心、复盘情绪、是否追高、是否扛单、是否报复性动作 |
| 系统一致性 | 行为是否和自己的规则一致 | 九型人格档案、复盘分数、是否系统内、每日训练 |
| 临盘克制 | 临场是否能慢一拍 | 行动前计划、临盘改计划、心境牵动、道场任务 |
| 主修兑现 | 当天训练是否真的回应人格对应的主修关卡 | 九型人格、主修关卡、关卡进度、每日训练卡、复盘 |

规则：

- 总分范围：20 到 98。
- 不允许与收益、交易结果、具体股票绑定。
- 人格未识别时，只能生成通用知行指数；人格识别后，指数必须绑定主修关卡。
- 每日训练卡必须反哺该人格对应的六大关卡，否则系统一致性和主修兑现分应下降。
- 趋势可以用于自我修行和教练/助教提醒，但不能用于投资宣传。

## AI观心规则

AI观心助手不是聊天机器人，不能输出买卖建议，只能做三件事：

1. 帮用户命名此刻心境。
2. 帮用户看见心贼和惯性动作。
3. 给出一个当下或明日可执行的修行动作。

标准输出结构：

- 触发人格
- 当前心贼
- 建议回到哪一关
- 今日只练一件事

AI回应必须读取上下文：

- 今日照心
- 九型人格档案
- 今日修行
- AI省察 / AI复盘
- 成长树当前关口
- 连续修行状态
- 道场任务
- 训练营教练/助教绑定状态

## 365天内容库规则

365天内容库是直播、视频号、小程序和AI观心教练的共同内容源。

每一天必须包含：

- 今日心证
- 戒律
- 训练
- 复盘

内容必须绑定六大关卡，不做行情判断，不做结果承诺。

## 双导师规则

系统采用双导师结构：

- 阳明先生：道统导师，作为心学语境和观心提醒，不代表历史人物本人言论。
- 训练营教练 / 助教：真人带练，只做修行陪跑、提醒和复盘监督。

真人带练可查看：

- 是否照心
- 是否完成每日修行
- 是否完成复盘
- 连续修行天数
- 知行指数趋势
- 道场任务完成情况

真人带练不可查看：

- 具体股票代码
- 具体买卖点
- 具体账户盈亏
- 任何可构成投资建议的数据明细

## 排行榜规则

排行榜只允许做修行榜，不允许做收益榜。

允许：

- 连修榜
- 省察榜
- 共修榜
- 知行进步榜
- 今日闭环榜

不允许：

- 收益榜
- 战绩榜
- 交易金额榜
- 具体标的表现榜
