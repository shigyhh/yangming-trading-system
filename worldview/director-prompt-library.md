# 导演提示词库

导演提示词库用于把世界观规则转成可拍、可讲、可展示、可训练的内容。它不是营销话术库，也不是行情解读库。

机器可调用数据：

- `data/director-prompt-library.js`

## 使用位置

- 视频号：每日心证短视频、人格短视频、关卡短视频。
- 直播：按人格、关卡、K线场景组织讲解。
- 小程序：每日修行卡、开盘照心、训练完成提示。
- AI教练：根据人格、关卡、场景生成观心回应。

## 总原则

- 先讲人心，再讲动作。
- 先命名心贼，再给训练。
- 每条内容只练一件事。
- 不评价具体标的，不暗示买卖方向。
- 结尾必须回到省察问题或训练动作。

## 内容结构

每条导演提示词必须包含：

- `stage`：所属六大关卡。
- `personality`：适配人格。
- `heartThief`：当前心贼。
- `klineScene`：触发场景。
- `trainingAction`：今日训练动作。

## 关卡内容方向

| 关卡 | 内容核心 | 结尾动作 |
| --- | --- | --- |
| 立志 | 先立边界，再入市场 | 写下今日不做清单 |
| 照心 | 不是预测市场，而是照见自己 | 命名当前心境 |
| 事上磨 | 触发来临时照规则做 | 完成一次微训练 |
| 破心贼 | 看见主人格旧反应 | 把旧反应改成新动作 |
| 知行合一 | 对照计划与行动差距 | 完成知行对照 |
| 致良知 | 无人提醒时仍能自省 | 写下明日自我提醒 |

## K线场景池

第一版场景池已内置在 `data/director-prompt-library.js`：

- 追涨
- 突破
- 回调
- 假突破
- 补仓
- 扛单
- 止损
- 踏空
- 浮盈回吐
- 连续亏损

每类场景都有：

- 6种结构。
- 适配关卡。
- 适配人格。
- 导演提示。

## 输出要求

任何内容输出都必须回答清楚：

1. 这个场景触发了什么心贼？
2. 对应哪一种人格惯性？
3. 应该回到哪一关修？
4. 今天只练哪一个动作？
5. 这一动作影响哪个知行指数维度？
