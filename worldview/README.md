# 阳明心学行为决策系统 Worldview

这个目录是产品世界观源文件。后续小程序、官网、直播内容、训练卡、AI提示词，都必须先服从这里的语言边界。

## 产品定位

上层定位详见：

- `worldview/global-worldview.md`

产品上层是“阳明心学行为决策系统 / 行为决策道场”。

交易是第一个应用场景，不是产品边界。

不要把这个产品做成金融工具。

它是交易者每天进入的修行空间，是一个围绕“知行合一”的每日修行工具。

我们不是帮助用户预测市场，而是帮助交易者在市场中照见自己。

适合：

- 打卡
- 照心
- 测评
- 每日任务
- 知行指数
- 复盘
- 陪伴

## 禁止词

- 暴富
- 翻倍
- 妖股
- 龙头
- 必赢
- 稳赚
- 财富自由

这些词会把产品带回金融刺激、结果诱导和短期欲望，不允许出现在小程序可见文案、AI报告、训练卡、直播标题和社群承接文案中。

## 文案方向

正确方向：

- 今天只练一件事。
- 开盘前，先照见此刻之心。
- 收盘后，只问哪一念带跑了自己。
- 让说过的计划，成为盘中的行动。
- 能不动，也是一种执行。

错误方向：

- 强刺激
- 强结果
- 强销售
- 强行情暗示
- 用金融热词制造焦虑

## 视觉方向

- 黑金
- 东方
- 克制
- 留白
- 呼吸感
- 黑里还有黑
- 金色只做方向和仪式感
- 主模块像法旨，不像信息卡

## 使用规则

新增页面、组件、训练卡、AI提示词前，先检查：

1. 有没有落在修行闭环里？
2. 有没有使用核心词强化世界观？
3. 有没有出现禁止词或结果诱导？
4. 有没有把用户注意力拉回“修自己”，而不是拉向外部行情？
