# Coze 最终版工作流收口配置

目的：把当前已经跑通的“AI诊断报告”升级为真正可用的最终版：

```text
开始
→ 自动评分代码节点
→ 生成诊断报告大模型节点
→ 结束
```

当前你已经跑通了“大模型生成报告”，但还不是最终版。最终版必须做到：

1. 输入用户答案 JSON。
2. 自动评分节点计算九人格分数、主人格、副人格、风险等级、高分子维度。
3. 大模型只读取自动评分结果，生成诊断报告。
4. 结束节点同时输出 `report` 和 `score_result`。

---

## 一、你当前截图里的两个问题

### 问题 1：score_result 是 null

原因：

```text
结束节点的 score_result 没有绑定到“自动评分代码节点.output”。
```

解决：

在结束节点里这样配置：

| 输出变量 | 绑定 |
|---|---|
| report | 生成诊断报告.output |
| score_result | 自动评分代码节点.output |

如果你当前画布里没有“自动评分代码节点”，那就说明现在只是 V1 报告生成器，还没进入最终版。

### 问题 2：报告里出现很多反斜杠

原因：

```text
大模型节点吃到的是被转义过的原始 input，而不是自动评分后的干净 output。
```

最终版里，大模型节点的输入不要再选开始节点的 `input`，要选：

```text
自动评分代码节点.output
```

---

## 二、最终版工作流结构

画布必须是：

```text
开始节点
→ 自动评分代码节点
→ 生成诊断报告大模型节点
→ 结束节点
```

不要让开始节点直接连到大模型节点。

---

## 三、开始节点配置

开始节点只保留一个字段：

| 字段 | 配置 |
|---|---|
| 变量名 | input |
| 类型 | String |
| 必填 | 是 |

输入内容是用户答案 JSON 字符串。

---

## 四、自动评分代码节点配置

节点名称：

```text
自动评分代码节点
```

输入变量：

```text
input = 开始.input
```

输出变量：

```text
output
类型：String
```

代码使用你已经跑通的那段完整代码即可。它的返回必须是：

```javascript
return {
  output: JSON.stringify(result, null, 2)
};
```

---

## 五、生成诊断报告大模型节点配置

节点名称：

```text
生成诊断报告
```

输入变量必须改成：

```text
input = 自动评分代码节点.output
```

不要再用：

```text
input = 开始.input
```

### 系统提示词

```text
你是“阳明心学交易人格 AI 诊断官”。

你的任务是根据自动评分结果，生成一份股民一看就懂、愿意认可、但不构成投资建议的交易行为诊断报告。

你必须遵守：
1. 不荐股，不预测行情，不承诺收益。
2. 不说“你这个人有问题”，只诊断交易行为模式。

5. 报告必须包含：测评说明、主人格、副人格、当前交易风险、最容易亏钱的场景、最该训练的一项能力、阳明心学提醒、7天行动建议、训练营分层建议。

合规边界：
```

## 六、结束节点配置

结束节点建议输出两个变量：

| 输出变量 | 类型 | 绑定 |
|---|---|---|
| report | String | 生成诊断报告.output |
| score_result | String | 自动评分代码节点.output |

这样前端既能展示报告，也能保存结构化评分结果。

## 八、最终版成功标准

试运行成功后，右侧结果应该有两个输出：

```text
report：完整诊断报告
score_result：完整评分JSON
```

`score_result` 不能是 null。

报告里不应该出现大量 `\"` 这种转义符。

报告应包含：

1. 主人格
2. 副人格
3. 风险等级
4. 九人格分数
5. 高分子维度
6. 亏钱场景
7. 训练能力
8. 阳明心学提醒
9. 7天行动建议
10. 训练营分层建议

## 九、再往后的真正产品版

这个最终版只是“评分 + 报告”工作流。

真正产品版还需要一个“抽题工作流”或外部网页：

```text
720题库
→ 随机抽27/45/90题
→ 用户答题
→ 提交 answers JSON
→ 调用本工作流
→ 展示报告
```

正式商业化建议：

```text
网页/小程序负责：抽题、答题、存数据
Coze负责：评分报告、AI解释、私域陪跑
```
