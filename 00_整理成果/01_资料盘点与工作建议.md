# 阳明心学交易系统资料盘点与工作建议

整理时间：2026-05-20

## 一、总判断

结论：现有资料基本满足“阳明心学交易系统”下一阶段的内容生产、直播转化、训练营搭建和讲师统一口径需要。

这批资料已经具备一个完整商业教育体系的雏形：

- 顶层定位：不是荐股体系，而是交易人格教育体系。
- 核心主张：交易即修心，盈利是人格稳定后的结果。
- 用户模型：九种交易人格、六级认证、成长地图。
- 内容素材：100句心学金句、短视频脚本模板、直播案例库。
- 转化工具：直播SOP、客户30问、私域引流、训练营方案。
- 产品方向：AI测评、AI行为矫正、交易人格训练营、讲师体系。
- 辅助资产：成长路径图、训练营模板、量化工具、PPT、XMind。

一句话判断：

你现在不是缺资料，而是缺“统一主线、统一文件结构、统一对外表达、统一内容排期”。

## 二、当前资料最强的五个部分

### 1. 品牌定位已经清楚

核心口径已经形成：

- 不是教你炒股，而是帮你建立稳定交易人格。
- 不是荐股直播间，而是交易人格修炼场。
- 不是暴利体系，而是长期主义、风控优先、知行合一。

这部分可作为所有直播、短视频、课程、私域成交的统一底层语言。

### 2. 九种交易人格已经能产品化

现有资料中已经出现：

- 冲动型
- 扛单型
- 完美主义型
- 偏执型
- 焦虑型
- 从众型
- 赌徒型
- 拖延型
- 平衡型

这些人格可以直接变成：

- 交易人格测评
- 九人格短视频系列
- 九人格直播专题
- 九人格训练营分层
- 九人格AI诊断报告

### 3. 直播转化体系已经成型

现有《A股直播话术SOP V2》《A股直播统一案例库》《人格破解手册》已经能支撑直播间落地。

推荐直播标准结构：

1. 情绪破冰：先让用户承认自己失控。
2. 痛点共鸣：追涨、扛单、满仓、焦虑、听消息。
3. 人格归因：不是技术问题，是人格问题。
4. 心学修正：致良知、知行合一、知止、克己。
5. 产品承接：测评、训练营、复盘、AI行为教练。

### 4. 短视频素材非常充足

已有材料可拆成至少 5 个内容系列：

- 阳明心学100句 × 交易短视频
- 九种交易人格短视频
- A股十大亏损场景短视频
- 交易人格觉醒短视频
- 直播切片金句短视频

### 5. 训练营方向已经具备雏形

已有：

- 21天训练营模板
- 30天知行合一交易训练营
- 良知复盘三问
- 阳明交易十戒
- 四格法
- AI交易修行系统

下一步只需要把它们整理成“每天一个主题、一个任务、一个复盘表”的执行版。

## 三、目前还缺的关键材料

### 1. 合规边界说明

建议补一份《阳明心学交易系统合规表达手册》。

必须明确：

- 不荐股
- 不承诺收益
- 不喊单
- 不展示夸张收益
- 不诱导贷款、融资、满仓
- 所有内容定位为交易认知、行为训练、风险教育

这是直播、短视频、私域成交最重要的保护线。

### 2. 用户转化链路还需要更细

建议形成标准链路：

短视频/直播间 → 交易人格测评 → 私域欢迎语 → 诊断报告 → 体验课 → 训练营 → 进阶课/认证体系。

每一步都需要独立话术和承接页面。

### 3. 课程交付还需要颗粒度

目前有总纲，但还需要：

- 每节课标题
- 每节课目标
- 每节课讲稿
- 每节课作业
- 每节课复盘表
- 每节课社群互动话术

建议优先做“30天训练营执行版”。

### 4. 案例库还需要真实化

现在案例结构很完整，但部分表达偏模板化。

建议补充：

- A股真实高频场景
- 用户原话
- 常见评论区留言
- 不同人格的真实账户行为
- 从亏损行为到训练结果的对比案例

### 5. AI产品还需要需求文档

已有 AI 交易人格测评、AI行为教练、AI复盘方向，但还需要：

- 测评题库
- 评分规则
- 报告模板
- 用户标签体系
- Agent提示词
- 产品页面原型
- 数据字段表
