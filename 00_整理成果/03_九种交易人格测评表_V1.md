# 九种交易人格测评表 V1

定位：这是一套面向普通股民的交易行为与心性觉察测评，用来识别用户在亏损、盈利、踏空、追涨、止损、持仓时最容易重复的行为模式。

合规说明：本测评只用于交易认知教育、行为觉察和训练分层，不构成任何投资建议、荐股建议或收益承诺。

## 一、用户填写说明

请根据你最近 3-6 个月真实交易状态作答，不要选“理想中的自己”，而要选“实际交易时的自己”。

每题 1-5 分：

| 分值 | 含义 |
|---|---|
| 1 分 | 完全不像我 |
| 2 分 | 偶尔如此 |
| 3 分 | 有时如此 |
| 4 分 | 经常如此 |
| 5 分 | 几乎就是我 |

建议开场话术：

> 这份测评不是给你贴标签，而是帮你看见自己在市场波动面前最容易失控的地方。交易亏损，表面是技术问题，深处往往是心性和行为模式问题。先看见自己，才谈得上真正改变。

## 二、45 道场景题

### A. 冲动型

| 编号 | 题目 |
|---|---|
| CX1 | 看到股票盘中快速拉升时，我很容易来不及分析就追进去。 |
| CX2 | 某只股票突然被很多人讨论时，我会担心错过机会，哪怕它原本不在我的计划里。 |
| CX3 | 连续错过几只上涨股票后，我下一次下单会明显变急。 |
| CX4 | 我经常买完之后，才开始找理由证明自己买得对。 |
| CX5 | 账户刚赚一笔后，我更容易加快交易频率，觉得自己“手感来了”。 |

### B. 扛单型

| 编号 | 题目 |
|---|---|
| KD1 | 明明股价已经跌破我原本设定的位置，我还是会告诉自己“再等等，会回来的”。 |
| KD2 | 亏损越大，我越不舍得止损。 |
| KD3 | 我常用“主力洗盘”“公司还不错”“长期肯定没事”来安慰自己继续拿着。 |
| KD4 | 我经常把短线做成中线，中线做成长线。 |
| KD5 | 我会提前写下止损位，但真正触发时又临时改口。 |

### C. 完美主义型

| 编号 | 题目 |
|---|---|
| WM1 | 我常因为想等更低的买点或更完美的位置，错过原本看好的机会。 |
| WM2 | 买入后只要没有马上按我的预期走，我就会觉得自己入场不够完美。 |
| WM3 | 我容易花很多时间比较方案，却迟迟不敢真正下决策。 |
| WM4 | 卖出后如果股票继续上涨，我会反复懊恼，觉得自己没有卖在最高点。 |
| WM5 | 我希望每笔交易理由都非常充分，否则就不敢执行。 |

### D. 偏执型

| 编号 | 题目 |
|---|---|
| PZ1 | 一旦我判断某个方向，很难听进相反观点。 |
| PZ2 | 股票下跌时，我反而更想证明自己看得比别人远。 |
| PZ3 | 我容易只寻找支持自己判断的信息，忽略风险提示。 |
| PZ4 | 别人提醒我该止损或该降低仓位时，我第一反应常常是反驳。 |
| PZ5 | 我对“市场可能已经变了”这件事接受得比较慢。 |

### E. 焦虑型

| 编号 | 题目 |
|---|---|
| JL1 | 买入后我会频繁刷新行情，哪怕只是很小的波动。 |
| JL2 | 持仓期间，我的情绪容易被分时线牵着走。 |
| JL3 | 我经常因为害怕亏损，过早卖掉原本计划持有的股票。 |
| JL4 | 股票一跌，我就想找人确认自己到底该不该卖。 |
| JL5 | 持仓会影响我的睡眠、工作或家庭状态。 |

### F. 从众型

| 编号 | 题目 |
|---|---|
| CZ1 | 看到群里、直播间或朋友都在买某只股票，我会忍不住跟着研究甚至下单。 |
| CZ2 | 别人说某只股票有机会时，我会比自己独立分析时更有信心。 |
| CZ3 | 市场恐慌时，别人一割肉，我也容易动摇。 |
| CZ4 | 我经常买入别人推荐的股票，但卖出时没人提醒我。 |
| CZ5 | 我更在意“大家怎么看”，而不是自己的交易规则。 |

### G. 赌徒型

| 编号 | 题目 |
|---|---|
| DT1 | 我常想通过一两笔重仓交易，把之前的亏损快速赚回来。 |
| DT2 | 连续亏损后，我更容易加仓或加大仓位。 |
| DT3 | 我对“翻倍”“涨停”“妖股”这类机会特别兴奋。 |
| DT4 | 我明知道风险很大，但还是会想“万一这次成了呢”。 |
| DT5 | 我有时会把交易当成刺激、翻身或证明自己的方式。 |

### H. 拖延型

| 编号 | 题目 |
|---|---|
| TY1 | 我知道应该复盘，但经常拖到忘记交易细节。 |
| TY2 | 我做过交易计划，但真正执行前总会临时犹豫或改变。 |
| TY3 | 我明知账户需要整理、仓位需要调整，却一拖再拖。 |
| TY4 | 我收藏了很多方法、课程和指标，却很少持续练完一个。 |
| TY5 | 我经常告诉自己“有空再说”，结果错过处理问题的窗口。 |

### I. 平衡型

| 编号 | 题目 |
|---|---|
| PH1 | 每次买入前，我基本知道自己的入场理由、止损位置和仓位上限。 |
| PH2 | 单笔亏损出现时，我能按规则处理，不会因为一笔交易否定自己。 |
| PH3 | 我不会因为别人赚钱，就随便改变自己的交易计划。 |
| PH4 | 我更关注一套方法长期是否有效，而不是一两笔输赢。 |
| PH5 | 我能在交易后复盘自己的行为，而不是只盯着盈亏结果。 |

## 四、计分规则

每一种人格 5 道题，每题 1-5 分，单项人格总分范围为 5-25 分。

| 维度 | 题目编号 | 计算方式 |
|---|---|---|
| 冲动型 | CX1-CX5 | 5 题相加 |
| 扛单型 | KD1-KD5 | 5 题相加 |
| 完美主义型 | WM1-WM5 | 5 题相加 |
| 偏执型 | PZ1-PZ5 | 5 题相加 |
| 焦虑型 | JL1-JL5 | 5 题相加 |
| 从众型 | CZ1-CZ5 | 5 题相加 |
| 赌徒型 | DT1-DT5 | 5 题相加 |
| 拖延型 | TY1-TY5 | 5 题相加 |
| 平衡型 | PH1-PH5 | 5 题相加 |

强度解释：

| 单项分数 | 解释 |
|---|---|
| 5-10 分 | 特征较弱 |
| 11-16 分 | 偶尔出现 |
| 17-21 分 | 特征明显 |
| 22-25 分 | 高度典型 |

主人格判定：

1. 先比较前 8 个风险型人格分数，最高分为主人格。
2. 如果平衡型分数 >= 21，且前 8 个风险型人格全部 <= 15，则主人格可以判定为平衡型。
3. 如果最高分有并列，优先选择最近 1 个月最影响交易结果的那个维度；报告中可写成“混合型倾向”。

副人格判定：

1. 前 8 个风险型人格中第二高分为副人格。
2. 如果副人格与主人格相差 <= 2 分，说明用户存在明显复合型模式。
3. 如果副人格 <= 12 分，可写为“副人格不明显，主要问题集中在主人格”。

当前交易风险判定：

| 条件 | 风险等级 |
|---|---|
| 主人格 >= 22，或两个风险型人格 >= 20，或平衡型 <= 10 | 高风险 |
| 主人格 18-21，或平衡型 11-14 | 中高风险 |
| 主人格 14-17，且平衡型 15-18 | 中等风险 |
| 所有风险型人格 <= 13，且平衡型 >= 18 | 相对稳定 |

平衡型分数说明：

| 平衡型分数 | 解释 |
|---|---|
| 5-10 分 | 缺少交易规则和情绪刹车 |
| 11-16 分 | 有觉察，但执行不稳定 |
| 17-21 分 | 已有稳定交易习惯 |
| 22-25 分 | 具备较强自控和复盘能力 |

## 五、报告生成结构

每份报告必须包含以下 7 个模块：

1. 主人格：最明显的交易模式。
2. 副人格：压力下容易暴露的问题。
3. 当前交易风险。
4. 最容易亏钱的场景。
5. 最该训练的一项能力。
6. 阳明心学提醒。
7. 7 天行动建议。

报告语气原则：

- 不说“你这个人有问题”，而说“你在这种市场情境下容易出现某种反应”。
- 不说“你不适合交易”，而说“你需要先修正这个重复亏损动作”。
- 不承诺收益，只强调认知、纪律、风控、复盘和行为训练。
- 让用户觉得“被看见”，而不是“被批判”。

## 六、九种人格结果库

### 1. 冲动型

主人格解释：

你最明显的交易模式是冲动型。你不是没有判断力，而是在行情快速变化、别人赚钱、股票突然拉升时，容易被“怕错过”的情绪推着走。你常常是先下单，再找理由。

副人格常见组合：

- 冲动 + 赌徒：容易追高、重仓、想快速翻身。
- 冲动 + 焦虑：买得急，买完又慌，容易来回打脸。
- 冲动 + 从众：看到别人买就心动，交易计划容易被外界带走。

当前交易风险：

中高到高风险。你的主要风险不是不会分析，而是下单前缺少停顿，容易把“机会感”误认为“确定性”。

最容易亏钱的场景：

热点突然拉升、直播间情绪高涨、朋友说某只股票要启动、自己连续踏空之后急于补票。

最该训练的一项能力：

下单前延迟能力。任何临时起意的交易，都必须先停 3 分钟，写清楚买入理由、止损位置和仓位。

阳明心学提醒：

心不定，手就乱。交易中的“知”不是知道机会来了，而是知道自己此刻是不是被贪念牵着走。

7 天行动建议：

| 天数 | 任务 |
|---|---|
| 第 1 天 | 记录最近 5 次冲动交易，写出当时触发你的情绪。 |
| 第 2 天 | 建立“下单前 3 分钟暂停”规则。 |
| 第 3 天 | 所有盘中临时想买的股票，只允许加入观察，不允许马上买。 |
| 第 4 天 | 写一张交易前检查表：理由、位置、仓位、止损。 |
| 第 5 天 | 复盘一次因为怕错过而买入的交易。 |
| 第 6 天 | 练习小仓位执行，不因盘中波动临时加仓。 |
| 第 7 天 | 总结自己的三大冲动触发词，例如“涨停、突破、别人赚了”。 |

### 2. 扛单型

主人格解释：

你最明显的交易模式是扛单型。你不是不知道止损重要，而是在真正亏损出现时，不愿面对“这笔交易可能错了”。你容易用等待代替规则，用希望代替行动。

副人格常见组合：

- 扛单 + 偏执：越跌越相信自己，越错越不愿认。
- 扛单 + 焦虑：心里很慌，但手上不动。
- 扛单 + 拖延：知道该处理，却总想明天再说。

当前交易风险：

高风险。小亏容易被拖成大亏，短线容易被动变长线，账户容易被一两只错误持仓拖住。

最容易亏钱的场景：

跌破原计划止损位、市场趋势转弱、个股利空出现、补仓后继续下跌。

最该训练的一项能力：

认错执行力。不是训练你预测更准，而是训练你在错的时候能及时停止损失扩大。

阳明心学提醒：

知错能改，才是真知。交易里真正的知行合一，不是嘴上知道止损，而是到了止损位能执行。

7 天行动建议：

| 天数 | 任务 |
|---|---|
| 第 1 天 | 列出账户里所有“原本不是打算拿这么久”的持仓。 |
| 第 2 天 | 给每只持仓写下重新持有的理由，不合格就列入处理清单。 |
| 第 3 天 | 设定每笔交易最大可承受亏损比例。 |
| 第 4 天 | 复盘一次从小亏拖成大亏的交易。 |
| 第 5 天 | 练习一次“按规则止损”的模拟或小仓位操作。 |
| 第 6 天 | 建立“不补情绪仓”规则：亏损后不能因为不服而补仓。 |
| 第 7 天 | 写下自己的止损宣言：我止损不是失败，是保护本金。 |

### 3. 完美主义型

主人格解释：

你最明显的交易模式是完美主义型。你总想买在最低、卖在最高、逻辑完全确认后再动手，但市场很少给人完美答案。你容易因为追求完美，错过合理机会，也容易在交易后反复懊恼。

副人格常见组合：

- 完美主义 + 拖延：想得多、改得多、真正执行少。
- 完美主义 + 焦虑：一买就担心自己买得不够好。
- 完美主义 + 偏执：过度相信自己的完整逻辑，不愿接受变化。

当前交易风险：

中等到中高风险。你的风险不是乱动，而是过度等待、过度比较、过度后悔，导致执行系统无法稳定形成。

最容易亏钱的场景：

看好的股票不给理想买点、卖出后继续涨、计划反复修改、因为一次没做到完美而否定整套方法。

最该训练的一项能力：

接受不完美的执行力。交易追求的是长期概率，不是每一笔都漂亮。

阳明心学提醒：

事上练，不是在脑子里等一个完美世界。真正的功夫，是在不完美的市场里守住原则。

7 天行动建议：

| 天数 | 任务 |
|---|---|
| 第 1 天 | 写下自己最常见的 3 个“再等等”理由。 |
| 第 2 天 | 把买点分成合理区间，而不是一个绝对点位。 |
| 第 3 天 | 用小仓位练习一次“达到条件就执行”。 |
| 第 4 天 | 复盘一笔因为等完美而错过的机会。 |
| 第 5 天 | 给每笔交易只允许设置 3 个核心条件。 |
| 第 6 天 | 练习卖出后不回头评价对错，只看是否符合规则。 |
| 第 7 天 | 写下自己的新标准：不求完美，只求一致。 |

### 4. 偏执型

主人格解释：

你最明显的交易模式是偏执型。你一旦形成判断，就容易把交易变成证明自己正确的过程。真正让你吃亏的，不一定是看错，而是不愿意承认市场已经给出相反证据。

副人格常见组合：

- 偏执 + 扛单：越错越拿，越跌越找理由。
- 偏执 + 赌徒：喜欢逆势重仓，试图证明自己比市场聪明。
- 偏执 + 完美主义：逻辑越完整，越难承认变化。

当前交易风险：

高风险。你容易把观点当成信念，把市场反馈当成噪音，最终让单笔错误不断扩大。

最容易亏钱的场景：

逆势抄底、利空后仍坚持原判断、别人提醒风险时反感、市场结构已经变化但自己仍按旧逻辑交易。

最该训练的一项能力：

反证能力。每次下单前必须写出“什么情况出现，说明我错了”。

阳明心学提醒：

致良知不是固执己见，而是能在事实面前放下私心。市场不负责证明你对，市场只反馈你是否顺势。

7 天行动建议：

| 天数 | 任务 |
|---|---|
| 第 1 天 | 找出最近一次“明明错了但不愿认”的交易。 |
| 第 2 天 | 每个观点后面写一句：如果出现什么信号，我就承认判断失效。 |
| 第 3 天 | 主动收集 3 条与自己观点相反的信息。 |
| 第 4 天 | 练习把“我认为”改成“目前证据显示”。 |
| 第 5 天 | 设定观点失效线，不允许用新理由无限续命。 |
| 第 6 天 | 找一笔历史交易，复盘自己忽略了哪些反证。 |
| 第 7 天 | 写下自己的交易戒律：市场永远比我大。 |

### 5. 焦虑型

主人格解释：

你最明显的交易模式是焦虑型。你不是没有计划，而是持仓后容易被波动牵着走。你怕亏、怕回撤、怕卖早、怕卖晚，最后不是股票在波动，而是你的心在波动。

副人格常见组合：

- 焦虑 + 冲动：买得急，买完又慌。
- 焦虑 + 从众：自己不确定，就不断找别人确认。
- 焦虑 + 扛单：心里很怕，但行动上又舍不得处理。

当前交易风险：

中高风险。你容易因为情绪波动做出过早卖出、频繁换股、反复确认、计划失效等动作。

最容易亏钱的场景：

持仓轻微浮亏、盘中快速震荡、市场消息密集、夜间刷到利空消息、别人观点与自己相反。

最该训练的一项能力：

持仓稳定力。把注意力从分时波动拉回交易计划。

阳明心学提醒：

此心不动，随机而动。不是市场不波动，而是你的心要先有定处。

7 天行动建议：

| 天数 | 任务 |
|---|---|
| 第 1 天 | 记录一天内自己看盘次数和情绪变化。 |
| 第 2 天 | 设置固定看盘时间，减少无意义刷新。 |
| 第 3 天 | 持仓前写清楚“什么波动不用管”。 |
| 第 4 天 | 练习把分时图切换为日线或计划表。 |
| 第 5 天 | 复盘一笔因为害怕而过早卖出的交易。 |
| 第 6 天 | 建立情绪评分：每次想操作前先给焦虑打 1-5 分。 |
| 第 7 天 | 写下自己的安定句：我只处理规则内的变化。 |

### 6. 从众型

主人格解释：

你最明显的交易模式是从众型。你容易被群聊、直播间、朋友观点、市场情绪带着走。别人看好时你心动，别人恐慌时你动摇，最后交易权不在自己手里。

副人格常见组合：

- 从众 + 冲动：别人一喊机会，自己马上跟进。
- 从众 + 焦虑：不敢独立判断，反复找人确认。
- 从众 + 扛单：买入靠别人，亏损后却只能自己扛。

当前交易风险：

中高风险。你最大的问题是缺少自己的交易标准，容易买在情绪热的时候，也容易卖在群体恐慌时。

最容易亏钱的场景：

听消息买入、群里集体看好、直播间高频刺激、朋友晒收益、市场恐慌时跟着割肉。

最该训练的一项能力：

独立决策力。任何外部信息都只能作为参考，不能直接变成买卖动作。

阳明心学提醒：

良知在自己心里，不在别人嘴里。交易不能把自己的方向盘交给市场噪音。

7 天行动建议：

| 天数 | 任务 |
|---|---|
| 第 1 天 | 列出最近 5 笔受别人影响的交易。 |
| 第 2 天 | 建立“别人推荐不当天买”规则。 |
| 第 3 天 | 每次听到消息后，必须写出自己的独立验证结果。 |
| 第 4 天 | 暂停一个最容易影响你交易情绪的信息源。 |
| 第 5 天 | 复盘一笔“别人让你买，但没人提醒你卖”的交易。 |
| 第 6 天 | 练习只按自己的计划处理一笔持仓。 |
| 第 7 天 | 写下自己的决策原则：信息可以听，按钮自己按。 |

### 7. 赌徒型

主人格解释：

你最明显的交易模式是赌徒型。你容易把交易当成快速翻身的机会，尤其在亏损后更想用一笔大的赚回来。你追求的不是稳定复利，而是强刺激和快速证明。

副人格常见组合：

- 赌徒 + 冲动：一兴奋就追，一不服就加。
- 赌徒 + 扛单：重仓做错后不愿止损。
- 赌徒 + 偏执：越逆势越敢赌，越亏越想证明自己。

当前交易风险：

高风险。你的核心风险是仓位失控和损失扩大，一次情绪化重仓可能抵消很多次正确努力。

最容易亏钱的场景：

连续亏损后急于回本、看到强势题材、朋友晒暴利、市场极端亢奋、自己不甘心认输。

最该训练的一项能力：

仓位控制力。先把每笔交易的最大损失锁住，再谈机会。

阳明心学提醒：

克己，是交易第一风控。能控制仓位的人，才是真正开始控制自己。

7 天行动建议：

| 天数 | 任务 |
|---|---|
| 第 1 天 | 统计最近 10 笔交易中最大仓位和最大亏损。 |
| 第 2 天 | 设定单笔交易仓位上限。 |
| 第 3 天 | 建立“亏损后 24 小时不加仓”规则。 |
| 第 4 天 | 复盘一次想翻身导致更大亏损的交易。 |
| 第 5 天 | 把“回本目标”改成“执行目标”。 |
| 第 6 天 | 练习用小仓位完成一次完整买卖闭环。 |
| 第 7 天 | 写下自己的风险底线：本金安全先于翻身冲动。 |

### 8. 拖延型

主人格解释：

你最明显的交易模式是拖延型。你知道很多道理，也收藏了很多方法，但真正该复盘、该执行、该处理持仓时，经常一拖再拖。问题不是你不懂，而是知和行之间断开了。

副人格常见组合：

- 拖延 + 完美主义：总想准备更充分，所以迟迟不开始。
- 拖延 + 扛单：知道该处理，但总想再等等。
- 拖延 + 焦虑：越焦虑越逃避，越逃避问题越大。

当前交易风险：

中等到中高风险。你的风险来自执行滞后，错过最佳处理窗口，让小问题拖成大问题。

最容易亏钱的场景：

该止损时没处理、该复盘时没记录、该建立规则时只收藏不执行、账户混乱却迟迟不整理。

最该训练的一项能力：

即时行动力。把大任务拆成当天能完成的最小动作。

阳明心学提醒：

知而不行，只是未知。交易里的改变，不从收藏方法开始，而从今天做一件小事开始。

7 天行动建议：

| 天数 | 任务 |
|---|---|
| 第 1 天 | 只做一件事：整理当前所有持仓。 |
| 第 2 天 | 建立 5 分钟复盘模板。 |
| 第 3 天 | 复盘最近一笔交易，不追求完整，只追求完成。 |
| 第 4 天 | 给每只持仓写一句处理原则。 |
| 第 5 天 | 删除或暂停 3 个让你分心的学习材料。 |
| 第 6 天 | 固定一个每日复盘时间，哪怕只有 5 分钟。 |
| 第 7 天 | 写下自己的执行句：先做最小一步。 |

### 9. 平衡型

主人格解释：

你最明显的交易模式是平衡型。你已经具备一定规则意识、仓位意识和复盘意识，不容易被单笔输赢彻底带走。你和大多数股民最大的区别，是开始把交易当成系统，而不是情绪反应。

副人格常见组合：

- 平衡 + 焦虑：整体有规则，但在波动加大时仍会心慌。
- 平衡 + 完美主义：规则意识较强，但有时对自己要求过高。
- 平衡 + 扛单：大部分时候稳定，但遇到特别看好的股票仍可能破戒。

当前交易风险：

相对稳定。但如果某个风险型人格分数也超过 18，说明你不是没有问题，而是已经有一定觉察和修正能力。

最容易亏钱的场景：

连续盈利后放松纪律、极端行情中被市场情绪带走、对熟悉模式过度自信、长期不复盘导致系统老化。

最该训练的一项能力：

系统迭代力。保持复盘，让稳定不是靠状态，而是靠机制。

阳明心学提醒：

知止而后有定。真正的稳定，不是没有情绪，而是有规则、有边界、有复盘。

7 天行动建议：

| 天数 | 任务 |
|---|---|
| 第 1 天 | 复盘自己最有效的一条交易规则。 |
| 第 2 天 | 找出最近一次差点破坏规则的场景。 |
| 第 3 天 | 优化自己的交易前检查表。 |
| 第 4 天 | 统计最近 10 笔交易是否按计划执行。 |
| 第 5 天 | 给自己的系统增加一条风险预警规则。 |
| 第 6 天 | 复盘一次盈利交易，重点看是否可复制。 |
| 第 7 天 | 写下下一阶段最需要保持的一条纪律。 |

## 八、报告开头与结尾模板

报告开头：

> 这份测评不代表你适不适合交易，也不是给你贴标签。它只是帮你看见：当市场波动、账户亏损、别人赚钱、自己踏空时，你最容易重复哪一种交易反应。看见这个反应，就是修正交易行为的第一步。

报告结尾：

> 你的问题不是没有救，也不是天生做不好交易。真正要修的，是那个在关键时刻总是重复出现的动作。阳明心学讲知行合一，交易中的改变不是知道更多，而是从下一笔交易开始，少犯一个重复错误。
