# 九种交易人格 AI 测评系统搭建 SOP

适用对象：阳明心学交易人格测评、Coze 智能体、私域诊断、训练营分层、AI 诊断报告。

核心原则：题库、抽题、计分要结构化；AI 只负责把结构化结果写成“客户听得懂、愿意认可”的报告，不让 AI 随意猜分。

合规说明：本系统只用于交易认知教育、行为觉察和训练分层，不荐股、不喊单、不预测行情、不承诺收益。

## 一、先定整体方案

### 最推荐架构

```text
720题库
→ 随机抽题器
→ 用户答题页
→ 自动评分器
→ 人格判定器
→ Coze/AI诊断官生成报告
→ 私域承接/训练营分层
```

### 三种落地方式

| 方案 | 做法 | 优点 | 缺点 | 推荐阶段 |
|---|---|---|---|---|
| Coze 纯智能体版 | 在 Coze 聊天里逐题问答、统计、生成报告 | 最快、零代码 | 多轮答题体验差，随机抽题和历史避重弱 | 临时演示 |
| 表单 + Coze 报告版 | 问卷星/飞书表单答题，Coze 根据分数生成报告 | 快速上线，稳定 | 随机抽题能力有限，数据分散 | 第一版 MVP |
| 独立网页 + Coze 报告版 | 网页抽题、计分、存数据，Coze 只负责生成报告和陪跑 | 最专业，最适合商业化 | 需要开发页面 | 正式产品 |

建议路径：

1. 先做“表单 + Coze 报告版”，验证话术、报告、转化。
2. 再做“独立网页 + Coze 报告版”，承接 720 题库、随机抽题、复测和训练营分层。
3. Coze 最终定位为“交易人格 AI 诊断官”和“训练营陪跑教练”，不要让 Coze 承担全部表单系统职责。

## 二、测评系统需要的模块

| 模块 | 功能 | 推荐实现 |
|---|---|---|
| 题库管理 | 存储 720 道题、人格、子维度、权重、题目层级 | Excel/数据库/飞书多维表格 |
| 随机抽题 | 从 720 道中抽 27/45/90 道 | 网页后端或 Coze 代码节点 |

| AI报告 | 生成诊断文字、7天建议、训练营推荐 | Coze 工作流大模型节点 |
| 数据沉淀 | 用户结果、来源、复测记录 | 数据库/飞书表格/CRM |

## 三、抽题规则

### 题量版本

| 版本 | 总题量 | 抽题规则 | 适合场景 |
|---|---:|---|---|
| 轻量版 | 27 | 每种人格 3 道 | 直播间、短视频引流 |
| 标准版 | 45 | 每种人格 5 道 | 私域首次测评，推荐默认 |
| 深度版 | 90 | 每种人格 10 道 | 体验课前测、训练营入营 |

### 抽题原则

1. 每种人格题数必须一样，避免偏分。
2. 每种人格内部尽量覆盖 4 个子维度。
3. 同一个用户复测时，优先避开最近答过的题。
4. 出题顺序必须打散，不连续出现同一人格。
5. 不同题量版本统一换算成百分制，便于横向比较。

### 百分制换算

```text
人格得分 = 实际得分 / 该人格满分 × 100

27题：每类3题，满分15分
45题：每类5题，满分25分
90题：每类10题，满分50分
```

## 四、评分规则

### 基础计分

每题 1-5 分：

| 分值 | 含义 |
|---|---|
| 1 | 完全不像我 |
| 2 | 偶尔如此 |
| 3 | 有时如此 |
| 4 | 经常如此 |
| 5 | 几乎就是我 |

### 主人格

前 8 个风险型人格中，百分制得分最高者为主人格：

- 冲动型
- 扛单型
- 完美主义型
- 偏执型
- 焦虑型
- 从众型
- 赌徒型
- 拖延型

### 副人格

前 8 个风险型人格中，百分制得分第二高者为副人格。

如果主人格和副人格差距小于等于 8 分，报告中提示“复合型倾向明显”。

### 平衡型

平衡型不和风险型人格直接混为一类解释。它代表用户的稳定能力、规则能力、风控能力和复盘能力。

特殊规则：

```text
如果平衡型 >= 84 分，并且全部风险型人格 <= 60 分，
则主人格可判定为平衡型。
```

### 风险等级

| 条件 | 风险等级 |
|---|---|
| 主人格 >= 88，或两个风险型人格 >= 80，或平衡型 <= 40 | 高风险 |
| 主人格 72-87，或平衡型 41-56 | 中高风险 |
| 主人格 56-71，且平衡型 57-72 | 中等风险 |
| 全部风险型人格 <= 52，且平衡型 >= 72 | 相对稳定 |

## 五、Coze 版本一：最快 MVP 做法

这个版本适合你马上跑私域，不用开发网站。

### Step 1：准备问卷

用现有自动评分 Excel 或问卷工具做 45 题标准版。

建议字段：

| 字段 | 说明 |
|---|---|
| 用户昵称 | 方便报告称呼 |
| 手机/微信 | 私域承接 |
| 来源渠道 | 直播/短视频/社群 |
| 45 道题答案 | 1-5 分 |
| 九人格分数 | 自动计算 |
| 主人格 | 自动判定 |
| 副人格 | 自动判定 |
| 风险等级 | 自动判定 |
| 高分子维度 | 自动判定或人工填写 |

### Step 2：创建 Coze 智能体

智能体名称：

```text
阳明心学交易人格 AI 诊断官
```

智能体介绍：

```text
基于九种交易人格测评结果，为股民生成交易行为诊断报告，帮助用户看见自己在追涨、扛单、焦虑、从众、重仓、拖延等场景中的重复亏损模式。
```

### Step 3：设置智能体人设

复制以下内容到智能体人设/提示词：

```text
你是“阳明心学交易人格 AI 诊断官”。

你的任务是根据用户的九种交易人格测评分数，生成一份股民一看就懂、愿意认可、但不构成投资建议的交易行为诊断报告。

你必须遵守：
1. 不荐股，不预测行情，不承诺收益。
2. 不说“你这个人有问题”，只诊断交易行为模式。
3. 语言要真实、克制、扎心但不羞辱。
4. 必须结合阳明心学的“知行合一、致良知、克己、知止、有定”。
5. 输出必须包含：主人格、副人格、当前交易风险、最容易亏钱的场景、最该训练的一项能力、阳明心学提醒、7天行动建议、训练营分层建议。
6. 报告要让用户感觉“被看见”，而不是“被批判”。

合规边界：
本报告只用于交易认知教育、行为觉察和训练分层，不构成任何投资建议、荐股建议、买卖建议或收益承诺。
```

### Step 4：创建报告生成工作流

工作流名称：

```text
生成交易人格诊断报告
```

开始节点输入参数：

| 参数名 | 类型 | 示例 |
|---|---|---|
| nickname | 字符串 | 老张 |
| test_version | 字符串 | 45题标准版 |
| main_type | 字符串 | 冲动型 |
| sub_type | 字符串 | 焦虑型 |
| risk_level | 字符串 | 中高风险 |
| scores_json | 字符串/对象 | 九人格分数 |
| sub_dimension_json | 字符串/对象 | 高分子维度 |
| channel | 字符串 | 直播间 |

工作流节点：

```text
开始节点
→ 数据整理节点
→ 大模型报告生成节点
→ 合规检查节点
→ 结束节点
```

### Step 5：大模型节点提示词

```text
请根据以下结构化测评结果，生成一份《九种交易人格 AI 诊断报告》。

用户昵称：{{nickname}}
测评版本：{{test_version}}
主人格：{{main_type}}
副人格：{{sub_type}}
风险等级：{{risk_level}}
九人格分数：{{scores_json}}
高分子维度：{{sub_dimension_json}}
来源渠道：{{channel}}

输出要求：
1. 开头先说明：本报告不是投资建议，不荐股、不预测、不承诺收益。
2. 用股民听得懂的语言解释主人格。
3. 解释副人格代表的压力下暴露问题。
4. 说明当前交易风险。
5. 写出最容易亏钱的 3 个场景。
6. 写出最该训练的一项能力。
7. 给出一句阳明心学提醒。
8. 给出 7 天行动建议，每天一个动作。
9. 给出训练营分层建议。
10. 结尾引导用户：先从一个重复错误开始修正。

语言风格：
- 真诚、克制、犀利但不羞辱。
- 多用“你在某种场景下容易……”而不是“你就是……”。
- 不要使用夸张收益、暴富、稳赚等词。
```

## 六、Coze 版本二：聊天内测评版

这个版本可以演示，但不建议作为正式测评。

流程：

```text
用户说“开始测评”
→ Coze 问用户选择 27/45/90 题
→ 工作流从题库中抽题
→ 每次输出 3-5 道题
→ 用户回复每题分数
→ 循环直到答完
→ 代码节点计分
→ 大模型生成报告
```

问题：

- 多轮答题容易中断。
- 用户回复格式不稳定。
- 45/90 题在聊天里体验较累。
- 历史避重和数据沉淀不如网页。

所以它适合做演示，不适合正式成交链路。

## 七、正式版：网页 + Coze

这是最推荐的长期方案。

### 页面结构

```text
首页
→ 选择题量：27/45/90
→ 答题页
→ 评分页
→ AI报告页
→ 私域承接页
```

### 首页文案

标题：

```text
九种交易人格测评
```

副标题：

```text
看见你在交易中最容易失控的地方
```

说明：

```text
这不是荐股工具，也不是收益预测。它只帮你看见：当市场波动、账户亏损、别人赚钱、自己踏空时，你最容易重复哪一种交易反应。
```

### 答题页

每题展示：

```text
题目：看到股票盘中突然直线拉升时，我很容易来不及分析就追进去。

1 完全不像我
2 偶尔如此
3 有时如此
4 经常如此
5 几乎就是我
```

### 结果页

必须展示：

- 主人格
- 副人格
- 当前交易风险
- 九人格分数条/雷达图
- 高分子维度

## 十一、建议下一步

优先做一个“标准版 45 题测评 MVP”：

1. 用 720 题库随机抽 45 题。
2. 每种人格固定 5 题。
3. 自动计分并生成主人格、副人格和风险等级。
4. 把结构化结果交给 Coze 生成报告。
5. 用报告承接私域和训练营分层。

跑通后再做 27 题轻量版和 90 题深度版。
