# 知行合一交易修行系统：产品闭环与直播内容

定位：不预测行情，不推荐股票，只训练交易者把“开仓前知道的规则”变成“盘中真正做到的行为”。

一句话：用阳明心学做骨架，把交易计划、情绪识别、行为复盘、每日训练和私域承接连成闭环。

## 一、真正有用的闭环

### 1. 立志：先定交易戒律

用户第一次进入系统，不先测评，先立戒律。

核心动作：

- 无计划，不开仓。
- 没有止损位，不开仓。
- 触发止损，不移动。
- 情绪失控，先离场。
- 亏损后，不用加仓赌回本。

直播可讲：

“交易系统的第一步不是教你买哪只，而是让你知道哪些动作绝对不能做。一个人只要能少犯三种错，交易结果就会完全不一样。”

### 2. 照心：开盘前识别贪、怕、急、执

开盘前 30 秒打卡，不是情绪日记，而是临盘风控。

核心问题：

- 今天是否急着回本？
- 今天是否害怕踏空？
- 今天是否想证明自己？
- 今天是否已经有明确计划？
- 今天是否适合下重手？

产品输出：

- 今日心境：平静、急躁、焦虑、贪婪、想回本。
- AI师父提醒：今日宜轻仓、少动、只守计划。

直播可讲：

“很多人以为自己是在看盘，其实是在被情绪看盘。开盘前先照心，是为了知道今天到底是谁在下单，是良知，还是私欲。”

### 3. 立计划：开仓前写下“所知”

开仓前必须写：

- 方向
- 入场理由
- 止损位
- 止盈/退出条件
- 计划仓位
- 当前情绪
- 不操作条件

系统原则：

没有出场，不谈入场。

直播可讲：

“你开仓前写不出止损，说明你不是在交易，而是在赌一个希望。真正的交易计划，必须先写最坏情况。”

### 4. 执行：盘中只看是否守界

产品不判断买卖对错，而判断用户有没有违背自己写下的规则。

重点记录：

- 是否临时追入？
- 是否移动止损？
- 是否亏损后加仓？
- 是否提前止盈逃跑？
- 是否因为群消息改变计划？

直播可讲：

“行情变化不可控，但你有没有移动止损、有没有冲动加仓、有没有提前逃跑，是可记录、可训练、可改变的。”

### 5. 复盘：平仓后对照“所行”

平仓后填写实际行为：

- 实际方向
- 实际仓位
- 实际止损
- 实际退出
- 触发情绪
- 哪条戒律被破

系统生成：

- 知行合一指数
- 纪律违背项
- 情绪触发点
- 本次扣分/加分
- AI师父点拨

直播可讲：

“赚了但破戒，也要扣分；亏了但守戒，也要加分。因为我们训练的不是一笔输赢，而是长期稳定的交易人格。”

### 6. 省察克治：每天只改一个动作

每日省察三问：

- 今天哪一笔不是行情打败我，而是私欲带跑我？
- 今天我犯的是贪、怕、急、执、赌、拖里的哪一种？
- 明天我只克治哪一个动作？

直播可讲：

“复盘不是找理由，是找私欲。今天只要找到一个重复犯错的动作，明天少犯一次，就是进步。”

### 7. 事上磨练：匿名历史K线逐根回放

训练不是猜涨跌，而是练纪律。正式版建议使用真实历史K线，但训练前必须匿名化：

- 隐去股票名。
- 隐去股票代码。
- 隐去日期。
- 隐去重大事件提示。
- 只保留纯盘面和成交/波动结构。

这样有两个好处：

- 安全：用户训练的是盘感和纪律，不是点评某只个股历史走势。
- 有效：避免用户认出某段经典行情，从记忆里作弊。

训练方式从“猜下一根涨跌”升级为“bar-by-bar逐根复盘回放”：

1. 系统先展示一段匿名历史K线的前几根。
2. 用户每放出一根，都要选择：买入、卖出、观望。
3. 系统记录每一次决策时的盘面位置。
4. 完成后揭晓后续走势、盈亏结果和纪律评分。
5. 可选择训练后再揭晓这段行情的真实来源。

训练题例：

“匿名历史片段 A-047。你原计划等待回踩确认。第8根出现急速拉升，量价明显放大。此刻你是买入、卖出，还是观望？”

数据更新：

- 后端维护 A 股离线K线库，小程序训练只读取匿名片段。
- 日K按交易日收盘后增量更新，补齐每只股票最后一根K线之后的缺口。
- 数据源支持 Eastmoney 原生接口；服务器安装 Python 环境后，可用 AkShare 增量补库；mootdx 可作为通达信协议备源。
- 盘中“近实时”训练不走日K，应单独缓存 30/60 分钟K或行情快照。

系统评分：

- 纪律得分
- 冲动识别
- 计划一致性
- 情绪触发
- 决策点盈亏
- 复盘后知行指数变化

直播可讲：

“训练场不是为了让你猜对下一根K线，而是让你在每一根K线刺激出现时，仍然能守住自己开盘前写下的话。真正的事上磨练，就是在盘面一根根走出来的时候，看见自己的贪、怕、急、执是怎么冒出来的。”

### 8. 测评驱动训练：对症修心

测评不能只是开场仪式，必须驱动后续训练内容。

训练片段按用户主问题加权抽取：

| 测评结果 | 加权训练片段 |
|---|---|
| FOMO/追高型 | 急速拉升后见顶回落、突破失败、冲高回落 |
| 扛单型 | 破位下跌后连续阴线、跌破支撑不反抽 |
| 焦虑型 | 横盘震荡、假突破、反复洗盘 |
| 赌徒型 | 连续大阳后的高波动、涨停打开、极端波动 |
| 拖延型 | 趋势转弱后迟迟不决、亏损扩大段 |
| 完美主义型 | 小亏不认、大盈早跑、等待完美买点错过 |
| 从众型 | 热门题材高潮段、群体情绪拥挤段 |
| 偏执型 | 单边走弱仍寻找利好解释的片段 |
| 平衡型 | 复杂震荡段和趋势跟随段，训练稳定执行 |

产品逻辑：

1. 用户先做九型交易人格测评。
2. 系统识别主人格、副人格、最高风险子维度。
3. K线训练库中的每个历史片段都打标签。
4. 系统按人格标签加权抽取训练片段。
5. 用户训练越多，系统越知道他在哪类盘面破功。
6. 后续AI报告不只基于测评，还基于真实训练行为。

直播可讲：

“同样一段行情，不同人格的人犯的错不一样。FOMO的人会追进去，扛单的人会不肯走，焦虑的人会来回割。我们的训练不是随机刷题，而是对症修心。”

## 二、产品模块结构

### 今日

功能：

- 开盘照心
- 今日戒律
- 今日直播讲点
- 今日修行任务
- 连续省察打卡

作用：

让用户每天打开一次，有轻动作、有内容、有提醒。

### 知行

功能：

- 知行合一指数
- 7日趋势
- 开仓前所知
- 平仓后所行
- 违背戒律统计
- 情绪触发识别
- 师父点拨

作用：

这是招牌功能。把“懂了做不到”变成可记录、可计算、可追踪。

### 训练

功能：

- 匿名历史K线逐根回放
- 情境选择题
- 九型交易人格测评
- 测评结果加权出题
- 省察日记
- 7天直播内容

作用：

支撑直播每天有主题，用户每天有动作。

### 我的

功能：

- 知行指数历史
- 交易戒律契约
- AI人格报告
- 训练记录
- 段位与积分
- 添加助理/进入社群

作用：

沉淀身份、数据和私域承接。

## 三、7天直播内容

### 第1天：无计划，不开仓

主题：立志。

讲法：

“交易最大的危险，不是不懂技术，而是每一笔都可以给自己找理由。今天先立第一条戒律：没有出场计划，不许入场。”

用户动作：

写下自己的第一条交易戒律。

### 第2天：移动止损，是心先动了

主题：知行裂缝。

讲法：

“你移动的不是止损线，是心里的侥幸线。止损位一动，说明你已经从执行计划，变成了和亏损讨价还价。”

用户动作：

复盘最近一次移动止损。

### 第3天：怕错过，是被行情牵着走

主题：致良知。

讲法：

“真正的机会不会因为你慢三分钟就消失。让你冲进去的往往不是机会，而是怕错过。”

用户动作：

记录一次 FOMO 冲动。

### 第4天：赚钱破戒，也要扣分

主题：交易人格。

讲法：

“靠破戒赚到的钱，会奖励你的坏习惯。系统不能只看盈亏，要看你是不是按计划做。”

用户动作：

找一笔赚钱但破戒的交易。

### 第5天：亏钱守戒，也是在进步

主题：事上磨练。

讲法：

“亏损不一定是失败。按计划亏，是交易成本；乱操作亏，是人格漏洞。”

用户动作：

写一笔亏钱但守纪律的交易。

### 第6天：复盘不是找理由，是找私欲

主题：省察克治。

讲法：

“真正的复盘不是证明自己没错，而是承认自己哪里被贪、怕、急、执带走了。”

用户动作：

完成每日省察三问。

### 第7天：用指数看见自己的变化

主题：知行合一指数。

讲法：

“盈亏曲线容易被行情影响，知行曲线反映的是你这个人有没有变稳。长期看，稳定人格才是交易底层能力。”

用户动作：

查看7天知行曲线，选出下周只克治的一个动作。

## 四、助理跟进话术

### 高风险用户

“你这次报告里最明显的问题不是技术，而是临盘冲动比较强。建议你先别急着加策略，先做7天‘无计划不开仓’训练，把知行指数稳定到70分以上。”

### 扛单用户

“你不是不会判断，而是不愿意承认判断错了。建议你今天先做一件事：写一笔最近移动止损的交易，我们帮你看是哪一种心魔在起作用。”

### FOMO用户

“你对机会很敏感，这是优点。但如果怕错过总是压过计划，就容易把机会做成风险。接下来7天重点练‘急涨不追，回踩再看’。”

### 平衡型用户

“你的自控力不错，接下来不要只追求少犯错，而要把稳定动作复制出来。建议你连续记录7天开仓计划，形成自己的交易戒律库。”

## 五、核心判断

这个系统必须始终围绕一句话：

“交易者最大的痛苦不是不知道，而是知道却做不到。”

所以产品不是测评工具，不是行情工具，也不是课程播放器，而是：

“把知行裂缝记录下来，把每一次破功变成下一次修行。”
