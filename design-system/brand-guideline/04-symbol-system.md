# 04 Symbol System

符号系统来自四个母题：

- 以交易照人心。
- 以复盘照行为。
- 以训练照变化。
- 以活镜照成长。

## 两套版本

| 版本 | 用途 | 复杂度 |
| --- | --- | --- |
| 品牌徽记版 | 海报、章节封面、课程卡片 | 可保留象形骨势 |
| UI Glyph 版 | 首页卡片、报告页、小程序 | 必须过 24px |

## 四个 Glyph

| Glyph | 概念 | UI 语义 |
| --- | --- | --- |
| trade | 交心 | 交易触发先照人心 |
| review | 回证 | 复盘让行为回到证据 |
| train | 重训 | 重复训练看见微变化 |
| growth | 镜生 | 长期记录人格松动 |

## 工程规则

- 24px 必须清楚。
- 16px 必须单独简化。
- 统一线宽。
- 统一视觉高度。
- 统一端点处理。
- SVG 使用 `currentColor`，方便适配深浅底。

## 禁止

- 使用图标库替代品牌 Glyph。
- 使用甲骨文表面风格。
- 加裂纹、印章、卦象、符咒。
- 让符号像交易信号或 K 线指标。
