# 05 Numeric System

交易训练一定会进入数字：天数、时间、编号、纪律分、完成度、复盘次数。数字系统的使命是建立信任，不是表达东方感。

## 规则

- 训练天数：`01 / 07`。
- 时间：`09:30`。
- 暂停动作：`10s`。
- 纪律分：`88%`。
- 报告编号：`REVIEW-20260602-01`。
- 表格数字优先等宽。

## 字体

当前使用 Noto Sans SC / 系统无衬线作为数字承载。后续原创字库最多加入 5% 的品牌骨架，不要过度设计。

## 视觉原则

- 数字要清楚。
- 百分号要轻。
- 斜杠、圆点、冒号保持同一节奏。
- 白底卡片里的大数字必须留足底部安全区。

## 禁止

- 用收益率语境包装纪律分。
- 用涨跌颜色暗示交易结果。
- 出现收益承诺、行情预测、买卖信号。
