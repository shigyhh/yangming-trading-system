# 03 Product UI Test

品牌必须进入真实产品流，而不是只停在展示板。

## 必测页面

| 页面 | 品牌资产 | 要验证的事 |
| --- | --- | --- |
| 首页 Hero | A1 / C16 / Display 标题 | 第一眼不是金融软件，而是交易行为训练系统 |
| 今日训练 | UI Glyph / 数字系统 | “停十秒”是可执行动作，不是口号 |
| 心证记录 | Text / Manifesto | 用户能写下第一念，不被诱导成交易建议 |
| 复盘报告 | A1 / Numeric / Risk Language | 报告像证据，不像收益单 |
| 风险边界 | Compliance Language | 不荐股、不喊单、不承诺收益、不预测市场 |

## 首页接入规则

- 导航使用 A1 或 C16，不再使用字体字形“照”当 Logo。
- 主标题保持“心不静，交易必乱。”。
- 右侧 AI 观心系统使用 UI Glyph 点到为止，不堆图标。

## 报告页接入规则

- 报告卡顶部使用 A1 作为品牌落款。
- 纪律分、训练天数、复盘时间使用 Numeric System。
- 所有报告语言围绕照见、觉察、训练、复盘。
- 不出现收益率、股票代码、买卖方向、行情预测。

## 白底测试

白底页面至少验证：

- 长文档。
- 复盘报告。
- 打印版。
- PDF 导出。
- 小字号风险提示。

白底成立，才说明品牌不是靠黑金氛围撑起来的。
