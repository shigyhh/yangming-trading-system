# 08 Zhao Character System

这一章只解决一个核心问题：如何把「照」做成阳明心学交易系统的唯一主形象艺术字，而不是把每个汉字都做成第二个 Logo。

## 总判断

主形象艺术字只用一个：照。

其他字可以设计，但它们只能成为字体系里的锚点字、模块符号和产品动作字，不能成为独立标志。世界级品牌不是每个字都抢眼，而是一个主符号极强，少数关键字有暗线，整个系统统一。

## 为什么是照

「照」不是一个结论词，而是一个动作词。交易之前，不是预测行情，不是喊单，不是马上判断对错，而是先照见自己。

它把阳明心学产品化成第一动作：

- 照见一念。
- 照见人心。
- 照见行为。
- 照见证据。
- 照见知而未行。

因此，「照」不是替代阳明心学，而是把阳明心学转译成交易训练入口。

## 字形原型

「照」的品牌结构采用：

- 上观：日、目、见、观。
- 中镜：镜线、停顿、界面。
- 下火：火、光、心火、念头痕迹。

这套结构让它既像汉字，又像符号；既有东方文字源头，又不复古；既有光感，又不玄学。

## 字系层级

| 层级 | 字 | 角色 | 使用边界 |
| --- | --- | --- | --- |
| 主字 | 照 | 品牌主标 / 最高识别符号 | Logo、App 图标、小程序头像、官网品牌位、报告封面、分享卡、视频片头、完成盖印 |
| 锚字 | 心、知、行 | 心学锚点字 | 大标题、章节标题、品牌宣言，不做独立 Logo |
| 法字 | 止、证、复、练 | 产品动作字 | 按钮、训练卡、复盘页、报告模块、课程模块 |
| 边界字 | 界 | 风险边界字 | 合规提示、风险教育、禁止事项，不做警告牌 |

## 使用规则

- 最高级入口只使用「照」：主 Logo、头像、封面、片头、品牌落款。
- `心 / 知 / 行` 只做 Display 锚点，不抢主标地位。
- `止 / 证 / 复 / 练` 只服务产品动作，不能被做成一组独立 Logo。
- `界` 只表达风险边界、合规边界和不越界。
- 同一个画面已有 A1 主标时，不再重复放大第二个「照」。
- 分享卡、报告页、短视频素材里的「照」必须使用原创矢量资产或统一组件，不得使用平台字体。

## 落地资产

| 资产 | 文件 | 用途 | 边界 |
| --- | --- | --- | --- |
| 主照印 | `web-next/src/components/brand/yangming-zhao-seal.tsx` | 完成盖印、分享卡水印、视频素材盖印 | 只服务「照」，不承载其他字 |
| 辅助字印 | `web-next/src/components/brand/yangming-character-mark.tsx` | 心、知、行、止、证、复、练、界的模块锚点 | 不能用于 Logo、头像、主标识 |
| 字系内容源 | `packages/content/brand-character-system.js` | 跨端共享字义、角色、顺序 | 页面不直接硬编码字义 |

辅助字印可以有格线、镜线、边界线等轻视觉处理，但必须弱于 A1 主标。它们不是第二套 Logo，而是产品路径里的章节字和动作字。

## 动效语法

动效也必须跟随字系层级，不能每页各自动。

| 字 | 动效 | 产品含义 | 使用边界 |
| --- | --- | --- | --- |
| 照 | 盖印 | 完成、照见、落款 | 只用于主照印、完成状态、分享卡水印和视频片头 |
| 心 | 微聚 | 第一念浮现，回到观心 | 测评入口、观心模块 |
| 知 | 显明 | 模糊变清楚，良知显现 | 解释、报告、知行提示 |
| 行 | 落位 | 动作落地，不停留在理解 | 行为训练、执行状态 |
| 止 | 停顿 | 停十秒，让念头露出来 | 停顿训练、临盘暂停 |
| 证 | 落章 | 心证落回证据 | 报告证据、复盘证据 |
| 复 | 回环 | 回看、回证、回到第一念 | 复盘页、循环之镜 |
| 练 | 沉淀 | 七日训练，长期塑形 | 训练完成态、成长谱 |
| 界 | 收束 | 风险边界、合规边界 | 风险教育、合规提示 |

实现规则：

- 动效定义放在 `packages/content/brand-character-system.js`，由组件统一读取。
- `YangmingCharacterMark` 自动读取心、知、行、止、证、复、练、界的动效。
- `YangmingZhaoSeal` 自动读取「照 / 盖印」动效。
- 动效必须慢、轻、克制；默认支持 `prefers-reduced-motion`。
- 不允许页面自己临时写一套和字义冲突的动画。

## 禁止

- 不把「心、知、行」做成第二个 Logo。
- 不做二十个形象艺术字，避免像字库展示。
- 不把「照」做成红章、符咒、卦象、佛教化图形或金融信号。
- 不使用财、富、赢、涨、牛等暴富暗示字。
- 不用悟、禅、道、空把系统带向玄学、宗教或灵性疗愈。

## 产品逻辑

先照，才有知。  
有知，才有行。  
行后，才有证。  
有证，才可复。  
反复，才成练。  
有界，才可信。

这就是阳明心学交易系统的字印闭环。
