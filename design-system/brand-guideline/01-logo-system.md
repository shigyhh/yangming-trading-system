# 01 Logo Final

Logo 系统分三层，不再用一个复杂字标硬撑所有尺寸。

## A1 主标

用途：

- 官网首页导航。
- 产品报告页。
- 课程封面。
- 白底文档封面。
- 正式品牌落款。

规则：

- 首先读成“照”，其次感到光。
- 中间镜线保持安静，不做祭坛感。
- 下方四点先是字的下盘，再是光的隐喻。
- 黑底、白底、反白都必须成立。
- 不加裂纹、红印、卦象、符咒或 K 线。

当前资产：

- Web SVG：`web-next/public/brand/yangming-a1.svg`
- React 组件：`web-next/src/components/brand/yangming-mark.tsx`

## B 精神符号

用途：

- 品牌源头页。
- 片头。
- 特殊章节。
- 徽章。

限制：

- 不做主导航常驻标。
- 不做 favicon。
- 不作为小程序头像主版本。

## C16 小标

用途：

- favicon。
- 小程序头像。
- App / H5 小尺寸角标。
- 24px 以下 UI 环境。

规则：

- 牺牲完整“照”字，保留记忆点。
- 上观、中镜、下火三层必须清楚。
- 16px 时不强求读完整汉字，只保证品牌残影。

当前资产：

- Web SVG：`web-next/public/brand/yangming-c16.svg`
- Next app icon：`web-next/src/app/icon.svg`

## 安全区

- A1 四周至少保留 0.75X。
- C16 四周至少保留 0.5X。
- 任何文字、边框、按钮不得贴近主标。

## 禁止

- 使用字体里的“照”字作为 Logo。
- 从商业字体转曲后修改。
- 用光效代替骨架。
- 用大红印章包装 Logo。
