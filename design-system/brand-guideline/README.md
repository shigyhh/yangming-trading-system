品牌宪法：

- One Word：照
- One Sentence：心不静，交易必乱。
- One Action：停十秒，先照心。
- One System：心证、取证、复盘、训练、纪律。
- One Boundary：不玄学，不预测市场。

最终判断：

以照为字，以心证为法，以训练为行，以风险为界。
